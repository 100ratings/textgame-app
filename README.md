# Painel Mestre PWA

Este projeto é uma versão otimizada e instalável (PWA) do seu Painel de Text Game.

## Como colocar no GitHub:

1.  **Crie um repositório novo** no seu GitHub (ex: `painel-mestre`).
2.  **Suba estes arquivos** para lá (`index.html`, `manifest.json`, `sw.js`).
3.  Vá em **Settings** (Configurações) do repositório.
4.  No menu lateral, clique em **Pages**.
5.  Em "Build and deployment", selecione a branch `main` e a pasta `/(root)`.
6.  Clique em **Save**.
7.  Em alguns minutos, seu site estará online em `https://seu-usuario.github.io/painel-mestre/`.

## Como instalar no celular:

1.  Abra o link do seu site no Safari (iOS) ou Chrome (Android).
2.  No iOS: Clique no botão de **Compartilhar** e selecione **Adicionar à Tela de Início**.
3.  No Android: Clique nos três pontinhos e selecione **Instalar Aplicativo**.
